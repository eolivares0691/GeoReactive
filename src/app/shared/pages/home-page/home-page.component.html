El Catálogo del Mundo

Core del Proyecto

Se utilizó la sintaxis moderna y las nuevas características de **Angular 19+**, aprovechando su potencial para crear aplicaciones reactivas y eficientes.

  • Framework de Diseño: Se implementó **DaisyUI** sobre Tailwind CSS para una interfaz de usuario limpia y personalizable.
  • Gestión de Datos: El proyecto implementa **rxResource** para la gestión de estado y peticiones HTTP, evitando la lógica duplicada.
  • Peticiones HTTP: La interacción con la API se maneja de forma flexible a través de **objetos, promesas y observables**, demostrando el manejo de asincronía.

API y Funcionalidad

Para el consumo de datos se utiliza la API REST Countries v3.1.

La búsqueda se activa por:

  • Escritura (con debounce).
  • Tecla "Enter".
  • Botón de búsqueda.

Tecnologías y Recursos Utilizados

Signals Services Resources y rxResources Reutilización de Componentes Peticiones HTTP Operadores de RxJS Debounce (effect) Cache Preservación de Estado (queryParams) Tailwind CSS DaisyUI
